## Task Management System

## Features

- **Add Task**
- **View Task**
- **Edit Tasks**
- **Update**
## How to run the project
clone the repository
```
  git clone https://github.com/sujinbabups/task-management.git
  cd task-management
```
run front end 
```
cd my-project
npm install
npm run dev
```
run backend
```
cd backend
npm i
node app.js
```
